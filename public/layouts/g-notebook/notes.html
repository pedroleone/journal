API rate limits

work

Keeping track of the limits I hit most often.


Figma REST API
100 requests/min per team. File reads are cheap; writes count more. Webhooks don't count against REST quota.


GitHub API
5,000 req/hr for authenticated requests. Search API is separate: 30 req/min. GraphQL has point-based limits — simple queries ~1pt, complex can be 100+.


OpenAI
TPM (tokens per minute) varies by tier. Tier 1 is tight on GPT-4. Batch API has 24hr window but much higher limits — good for non-realtime jobs.


General rule: build in exponential backoff from the start. Don't treat rate limits as edge cases.
